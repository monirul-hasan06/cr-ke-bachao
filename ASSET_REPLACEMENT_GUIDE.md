# Asset Replacement Guide

To make the game fully realistic-cartoon, replace the placeholder assets with professional game assets using the same filenames.

## Character style

Use:

- Realistic cartoon mobile action game style
- Bangladeshi university/student look
- 3/4 top-down camera angle
- Transparent PNG/WebP
- Clean floor shadow
- Consistent character proportions
- No gore details
